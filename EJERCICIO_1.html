HTML

HTMLson las siglas de HyperText Markup Language, que en español significa Lenguaje de Marcado de Hipertexto.

Es el lenguaje de marcado predominante para la creación de páginas web. es muy facil ver el codigo HTML de una pagina web, la opcion exacta cambia de un navegador a otro, pero normalmente se encuentra en el menu que aparece al pulsar el boton derecho del raton.

HTML se compone de etiquetas que se escriben entre los simbolos menor que y mayor que

Historia del HTML

Los inicios del lenguaje HTML se remontan a 1989, cuando Tim Berners-Lee, un científico del CERN (Organización Europea para la Investigación Nuclear), propuso un sistema de gestión de información basado en hipervínculos.

Versiones del HTML

Tim Berners-Lee definio la primera version de HTML en el año 1991

En la actualidad, la ultima version de HTML es la 5, que fue publicada en octubre de 2014.
